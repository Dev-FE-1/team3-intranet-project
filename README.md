### 프로젝트 정의

---

삼조트라넷은 임직원들에게 **필요한 정보**를 효율적으로 제공하고, 편리한 인사 관리를 지원하는 것을 목표로 합니다.

사용하기 쉽고 직관적인 인터페이스를 제공하여, 임직원들이 상황에 필요한 자료와 도구를 빠르게 사용할 수 있도록 도와줍니다.
